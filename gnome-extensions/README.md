# How to use

List of extensions to be installed:

1. AppIndicator and KStatusNotifierItem Support
2. Just Perfection
3. Space Bar
4. Switcher
5. Tactile
6. TopHat

```bash
# save
$ dconf dump /org/gnome/shell/extensions/ > extensions.txt
# load
$ dconf load /org/gnome/shell/extensions/ < extensions.txt
```
